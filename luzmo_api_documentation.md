# Luzmo API Documentation

## Main Developer Docs

**URL**: [https://developer.luzmo.com/](https://developer.luzmo.com/)

This includes:

- **Core API** - Build datasets & dashboards, manage users & groups
- **Embedding** - Libraries for integrating dashboards in your apps
- **Plugin API** - Connect your own data stores or APIs
